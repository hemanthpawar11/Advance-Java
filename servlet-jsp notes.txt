Servlet

Servlet is technology which is used to develop web applications.
Servlet is an API that provides many interfaces and classes.
Servlet in java is a class that runs on web server.
Servlet class will acquire property of server creating response and reading request sent by client.
All classes related to servlet present in javax.servlet package.
-----------------------------------------------------------------------
What is Server?
Server is a computer that provides service to other computer.
Server accepts the requests sent by the client, process the request and sends back the response to the client.

What is Web-Application?
Any application that require internet and run on server.

HttpServlet: 

The HttpServlet is an abstract class present in javax.servlet.http package
In HttpServlet there are no abstract method but still they have made HttpServlet class
as an abstract class because ,to avoid unwanted object creation directly. 
To create servlet class we were generally recommend to extend HttpServlet class.
HttpServlet class is a subclass of GenericServlet where service method is overriden.
 
For handling request sent by the client HttpServlet class provides following methods:

1.service(ServletRequest req,ServletResponse res) 
2.service(HttpServletRequest req, HttpServletResponse res) 
3.doget(HttpServletRequest req, HttpServletResponse res) 
4.dopost(HttpServletRequest req,HttpServletResponse res) 
5.dodelete(HttpServletRequest req,HttpServletResponse res)


This method accepts 2 parameters:
1.HttpServletRequest object-it stores the data sent by the client through the request.
2.HttpServletResponse object-it stores the response of the servlet that needs to be sent to the client.

RequestDispatcher

RD is an interface present in javax.servlet package.

Use: To dispatch the request from one servlet to another servlet or one servlet to another jsp/html pageor from onr html to another html page.

forward() :it forwards the request from one servlet to another resource(servlet,html,jsp) on the server.

include()-it includes the content of a servlet in another resource.

How does servlet works: 
Servlet is a small piece of program that run on the server ,now 
with any device we send the request to the server 
Then server will sends the request to the servlet Container(component present in server) ,where 
servlet container will process the request and its going to trigger proper 
servlet class present inside the server,and inside servlet classes we will 
be having doGet() where that method is invoked and accept the request 
and sends back the response.
Servlet container(it will process the url of request

JSP stands for java Server Pages
It is a technology which is used to create web applications similar 
to servlet.
JSP can be considered as an extension to servlets as it provides some 
extra functionalities than servlet.
It is advanced version of servlet.
A JSP file should be saved with .jsp extension 
A JSP file consists of html tags and jsp tags.
Hence jsp is a technology which allows java code inside html body.

Advantages of jsp over Servlets
JSP is extension of servlet so we can use all features of servlet in JSP.
JSP provides implicit objects which makes application development easy.
JSP file can be managed and maintained easily because we can separate 
development code and design code.But in servlet we mix development code 
and design code.
JSP provides many tags to write java code,these tags are easy to use and
understand.
The use of jsp tags and implicit objects reduce the length ofthe code.


JSP Tags
In JSP,java can be written in html body using following tags:
1.Declaration tag-used to declare variable and method.
	Syntax:<%! declare variable/method %>
	Ex:<%!  int a=20;
				public in test() {
					
					}
		%>
	Everything written in declaration tag will be present outside the service method.
		Considered as global variables,available throughtout JSP page.

2.Expression Tag-used to writing the output statements .So we dont have to use print()
		if we are using this tag.
	Syntax:<%=  output statements %>
	Ex:<%=   "sum is "+ (10+20) %>

3.Scriplet Tag-used to execute any java code in JSp.
		Everything written in this tag will be present inside the service method.
			Syntax:<% any java code %>
			Ex:<%  int a=20;
					int b=30;
					System.out.print(b-a);
		%>
