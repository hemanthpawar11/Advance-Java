* in jdbc 5 steps
 * in hibernate 3 steps : javax.persistence pkg
 * we have create 3 objects
 * all 3 are interface
 * 1.EntityManagerFactory--1st and 2nd step
 * call createEntityManagerFactory("vikas") : present in Persistence class
 * static method
 * String argument : persistence unit name(find it in persistence.xml file)
 * give object of EntityManagerFactory
 * 
 * 2.EntityManager----3rd step
 * call createEntityManager() : present in EntityManagerFactory
 * non static method
 * give EntityManager object
 * inbuilt methods to perform crud operation
 * insert--persist()
 * update--merge()
 * delete--remove()
 * fetch---find()
 * customized query---createQuery()
 * 
 * 3.EntityTransaction---4th and 5th step
 * call getTransaction() : present in EntityManager
 * non static method
 * 2 imp method
 * begin()----start the operation
 * commit()---save the operation done
 * 
 * 
 * Entity Class : hibernate create table
 * Rules to create an Entity class
--class should be annotated with @Entity
--entity class must have one primary key( id )to a row  for easy identification of objects  and annotated with @Id 
--class must be public non abstract class.
--class should have public zero parameterized constructor or public default constructor.
--for every variable of entity class there should be public getter and setter methods.
